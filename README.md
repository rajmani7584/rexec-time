# rexec-time
Get Program Execution time

### Download from [here](https://github.com/rajmani7584/rexec-time/releases/latest)

# Usage

```
Usage: rexec-time [args] [command] [command-args]
Options:
  -h, --help             Show this help message and exit
  -v, --version          Show program version and exit
  -s, --seconds          Display execution time in seconds
  -ms, --miliseconds     Display execution time in miliseconds (default)
  -us, --microseconds    Display execution time in microseconds
  -ns, --nanoseconds     Display execution time in nanoseconds
This program executes commands and displays execution time.
```
